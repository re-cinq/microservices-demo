# Implementation Plan: Frontend Product Search Filter

**Branch**: `main` | **Date**: 2026-05-11 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/002-product-search-filter/spec.md`

## Summary

Shoppers on the Online Boutique home page have no way to narrow the product grid without scrolling through every item. This plan delivers a browser-side search box that filters visible product cards by name on every keystroke — using product data already in the DOM, with a single template file change and no backend, infrastructure, or dependency modifications.

## Phase 0 — Research

See [research.md](research.md) for full decision log. Summary of the six resolved decisions:

| # | Decision | Chosen approach |
|---|---|---|
| 1 | Filter trigger event | `input` event (covers keyboard, paste, cut, browser clear button) |
| 2 | Show/hide mechanism | Toggle `element.style.display` (`''` / `'none'`) |
| 3 | Match algorithm | Case-insensitive `indexOf` substring match |
| 4 | Name data source | `data-product-name` attribute stamped by Go template at render time |
| 5 | No-results element | Pre-rendered hidden `<div>`; toggled by JS |
| 6 | Accessibility | `type="search"` + `aria-label`; keyboard-operable |

### Implementation Sketch

The entire implementation fits in one edit to `src/frontend/templates/home.html`:

1. **Search input** — inserted in the `col-12` row above the `{{ range $.products }}` loop:
   ```html
   <input id="product-search" type="search" class="form-control"
     placeholder="Search products..." aria-label="Search products" autocomplete="off">
   ```

2. **Data attribute** — added to every `.hot-product-card` div:
   ```html
   data-product-name="{{ .Item.Name | html }}"
   ```

3. **No-results element** — inserted after the `{{ end }}` of the product loop:
   ```html
   <div id="product-no-results" style="display:none;">
     <p class="text-muted">No products match your search.</p>
   </div>
   ```

4. **Filter script** — IIFE inserted after the no-results element:
   ```js
   (function () {
     var input = document.getElementById('product-search');
     var cards = document.querySelectorAll('.hot-product-card');
     var noResults = document.getElementById('product-no-results');
     input.addEventListener('input', function () {
       var query = this.value.trim().toLowerCase();
       var visible = 0;
       cards.forEach(function (card) {
         var name = (card.getAttribute('data-product-name') || '').toLowerCase();
         var show = !query || name.indexOf(query) !== -1;
         card.style.display = show ? '' : 'none';
         if (show) visible++;
       });
       noResults.style.display = (visible === 0) ? '' : 'none';
     });
   })();
   ```
